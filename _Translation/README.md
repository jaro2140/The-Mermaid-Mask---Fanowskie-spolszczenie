# _Translation

Publiczne źródła tłumaczenia.

Ten folder może zawierać pliki CSV, JSON, YAML lub inny format używany przez daną grę.
Zalecany układ:

- `source/` - eksport tekstów źródłowych bez oryginalnych assetów gry.
- `pl/` - polskie tłumaczenie gotowe do budowania patcha.

Zasady:

- nie zmieniaj identyfikatorów, kluczy i nazw technicznych,
- zachowuj zmienne, znaczniki i znaki nowej linii,
- większe pliki generowane automatycznie trzymaj w prywatnym repo, jeśli nie są
  potrzebne publicznie.
