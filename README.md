# The Mermaid Mask - Fanowskie spolszczenie

🇵🇱 Fanowskie spolszczenie do gry **The Mermaid Mask** (SFB Games).

Nieoficjalne polskie tłumaczenie gry **The Mermaid Mask**.

Projekt wymaga legalnie posiadanej kopii gry. Repozytorium publiczne zawiera tylko
pliki potrzebne graczom i testerom: instalatory, gotowe patche oraz źródła
tłumaczenia. Nie zawiera oryginalnych plików gry ani prywatnych narzędzi
developerskich.

## Struktura repozytorium

- `_Installers/` - skrypty instalacji, weryfikacji i przywracania oryginału.
- `_Patches/` - miejsce na pliki patcha publikowane w Releases.
- `_Translation/` - publiczne pliki tłumaczenia (źródła EN i gotowe PL).

## Instalacja

1. Pobierz najnowszą paczkę z zakładki Releases.
2. Rozpakuj paczkę patcha do folderu wskazanego w instrukcji wydania.
3. Uruchom instalator odpowiedni dla systemu (na razie tylko SteamOS/Linux).
4. Uruchom grę, w Opcjach wybierz język "Polski".

Szczegółowe komendy instalacji zostaną dopisane po przygotowaniu pierwszego
publicznego wydania patcha.

## Zgłaszanie błędów

Zgłaszając błąd, podaj:

- wersję gry,
- wersję patcha,
- system operacyjny,
- miejsce w grze, w którym występuje problem,
- zrzut ekranu lub dokładny tekst błędu, jeśli to możliwe.
